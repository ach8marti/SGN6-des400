Through simulated group chats, and hidden evidence, players analyze conversations, collect clues, and question suspects to uncover the truth.

**_How to launch the game:_**

As of now the game is not *yet deployed* on the web. Steps to run the game is as follow:
1. Download the files in this repository.
2. Use terminal/command prompt: one for frontend and one for backend
3. To start *backend*: cd to the **"backend" folder**, then use **"npm start"**
4. To start *frontend*: cd to the **"frontend" folder**, then use **"npm run dev"**
5. **Link**, *http://localhost:5173*, will appear on the terminal running the frontend scripts
6. *Ctrl+Click* on the link to open the game
7. To close the game, close the browser and the two terminals.
